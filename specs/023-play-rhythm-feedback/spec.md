# 023 游戏节奏与反馈闭环

## 背景

当前版本已经具备 25 个关卡、五类糖果球和多种场景机关，但实玩时仍有三个直接影响儿童体验的问题：空射或球已经停下后仍固定等待约 5.2 秒；命中、得分、连击和糖果狂热文字会在同一时刻堆叠；飞行技能只有符号，没有明确名称。存档中已经存在声音开关，但游戏没有实际声音反馈。

本轮目标是先修复玩法节奏和反馈层级，再补足轻量音效与必要的关卡宽容度，让孩子可以连续玩 3 关而不被等待、误结算或不清楚的交互打断。

## 已确认方案

采用“核心节奏闭环”方案，不进行全量玩法重写：

1. 发射后按球的真实运动状态结束回合。全部球出界时快速收尾；全部球持续低速一段时间后收尾；只有持续运动或异常状态才使用最大时长兜底。
2. 回合收尾与下一球生成分离，保留短暂的命中观察时间，但从球停止到下一球可操作不超过 1.2 秒。
3. 得分和连击共用一个聚合提示，同一批碰撞只保留一个动态数字；糖果狂热作为高优先级庆祝，不再与普通反馈争抢顶部区域。
4. 飞行技能按钮同时显示图形和短名称，并用环形进度表达剩余可用窗口。技能使用后立即切换为已使用状态并退出交互。
5. 使用 Web Audio 在本地程序化生成发射、命中、连击、技能、过关和失败短音效，不引入网络资源。声音偏好继续使用现有 `soundEnabled` 存档字段。
6. 只调整实玩中明显阻塞 1-3 发通关目标的前 25 关参数，不改变关卡编号、解锁记录或存档格式。

## 验收标准

- 连续完成或重玩 3 个关卡，无卡死、误结算、重复生成球或无法继续。
- 有效命中后 150ms 内出现可见反馈，并在声音开启时播放对应短音效。
- 球停止后最多 1.2 秒可以拖动下一颗球。
- 球仍高速运动、分裂球仍有任一子球运动时，不提前结束回合。
- 普通关卡的主要目标尽量允许玩家在 1-3 发内完成，剩余球数作为儿童容错。
- 静音设置刷新页面后仍然保留。
- 桌面和 iPad 纵向视口中，HUD、技能按钮和庆祝文字不重叠。

## 非目标

- 不增加账号、广告、内购、排行榜、埋点或服务端。
- 不修改 PWA 存档版本、GitHub Pages 部署方式和已解锁进度。
- 不新增章节或扩展 25 关之外的内容。
- 不在本轮整体拆分 `GameScene`；只抽取有独立规则和测试价值的回合状态判断与音效模块。
