# 024 关卡目标与三星冲刺

## 背景

当前版本已经解决固定回合等待、反馈堆叠、技能辨识和声音缺失，但游戏过程中只显示分数、剩余球和糖果连锁。玩家看不到目标完成度，也不知道距离下一颗星还有多远；结果页只有总分和星级，缺少“一发制胜”“新纪录”或失败差距等继续游玩的动力。

本轮围绕成熟休闲游戏常见的短目标闭环进行优化：开局即可看到本关目标，每次清除目标都更新完成度，分数跨过星级线时立即庆祝，结算页用一句话总结本局表现。

## 已确认方案

1. 游戏 HUD 增加目标进度和三星进度条，星级阈值直接使用关卡现有 `starScores`。
2. 三星进度条使用深色轮廓、明暗填充、斜纹和星形标记，不只依赖颜色区分。
3. 分数一次跨过一颗或多颗星时，只显示一条最高档位庆祝，并同时点亮所有新获得的星。
4. 独立统计真实发射次数；糖果狂热奖励球只改变剩余球数，不影响发射次数。
5. 结果页展示“一发制胜”“N 发过关”“本关新纪录”或“还差 N 个目标”，不新增额外计分规则。
6. 保持 25 个关卡、星级阈值、物理参数、存档版本和部署方式不变。

## 验收标准

- 进入任意关卡后，目标完成度与三星进度始终可见且不遮挡现有 HUD。
- 每次目标清除后，目标计数立即更新。
- 分数跨越星级阈值后 150ms 内出现点亮动画和文字反馈。
- 一次从 0 星跨到 2 星或 3 星时，不创建多条重叠庆祝文字。
- 糖果狂热奖励额外球后，结果页仍显示真实发射次数。
- 胜利页能区分一发制胜、普通过关和新纪录；失败页明确显示剩余目标数。
- 桌面和 iPad 834x1194 纵向视口中无文字或进度条重叠。
- 连续游玩 3 关无卡死、重复结算或错误解锁。

## 非目标

- 不新增球种、机关、章节或第 26 关以后的内容。
- 不修改计分公式、星级阈值、关卡目标耐久度或可用球数。
- 不新增账号、广告、内购、排行榜、埋点或后端。
- 不升级 `GameSave.version`，不迁移 `tantanle-save-v1`。
